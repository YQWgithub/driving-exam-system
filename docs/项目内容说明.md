# 项目内容说明

## 项目概览
- 项目名称：驾驶证模拟考试系统（`driving-exam-system`）
- 技术栈：
  - 后端：Spring MVC + MyBatis + Maven（多模块）
  - 前端：Vue 3 + Vite + Element Plus + Axios
  - 数据库：MySQL

## 目录结构
- `driving-exam-entity`：实体与通用对象
- `driving-exam-dao`：Mapper 接口与 MyBatis XML
- `driving-exam-service`：业务服务层
- `driving-exam-web`：Web/Controller 层（REST API）
- `frontend`：前端工程
- `database`：数据库初始化与题库脚本
- `docs`：项目文档

## 主要功能状态
- 用户登录/注册：已实现并可用
- 章节练习：已实现并可用（依赖题库数据）
- 模拟考试：已实现并可用（开考、答题、交卷）
- 错题本：已实现并可用（错题收录、查询、移除）
- 成绩查询：页面与接口已接通，可查看记录
- 后台管理：管理员可访问 `/admin`，包含控制台、题库管理、用户管理、系统配置

## 启动方式（当前）
- 后端：`driving-exam-web` 通过 `tomcat7:run` 启动，默认端口 `8083`
- 前端：`frontend` 通过 `npm run dev` 启动，默认端口 `5173`
- 代理：前端 `/api` 代理到 `http://localhost:8083/driving-exam`
